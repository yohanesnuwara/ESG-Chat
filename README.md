# ESG-Chat

<img src="https://user-images.githubusercontent.com/51282928/230783659-fbe9b43f-01d2-48fd-84d5-eefd093fb628.png" alt= "esg-chat" width="700">

ESG-Chat is a programmatic chatbot that can answer queries related to ESG (environmental, social, and governance) or sustainability topics. It's like ChatGPT for ESG. For corporates, being engaged in sustainability is important. We can use this chatbot to give us advice and strategy on how to improve sustainability.
